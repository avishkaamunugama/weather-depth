The application requires a Python environment with Open3D installed. Use the following command if not already installed.
- pip install open3d

Before running the application make sure the MacOS application has PythonKit framework linked, if the application is missing PythonKit or show any warnings, please add the PythonKit library using Swift Package manager. 
- .package(url: "https://github.com/pvieito/PythonKit.git", .branch("master"))

For further support on setting up PythonKit, please refer the instrucntions provided at "https://github.com/pvieito/PythonKit.git".
